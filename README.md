# DataScience | Wifi
## [데이터사이언스] 공공 Wi-Fi 설치와 효율성 분석 | 인구 밀도와 인터넷 접속률과의 상관관계
> 공공 Wi-Fi의 설치 수와 사용량, 인터넷 사용 시간, 지역별 인구 밀도 상관관계를
> EDA 기법과 시각화(선그래프/막대/상자그림/트리맵/지도/버블차트 등)를 통해 분석한 결과를 포함합니다.

---

## 데이터셋
- 서울특별시_공공와이파이 AP별 사용량 : AP별 GB 단위 사용량
- 서울특별시_공공와이파이 서비스 위치 정보 : 관리번호, 주소, 좌표, 설치년도 등
- 서울시 인터넷 이용빈도 및 주 평균 인터넷 이용시간
- 서울특별시 인구밀도(구별)

---

## 분석 결과 요약
- 공공 Wi-Fi가 서울 전역에 널리 분포하나 설치 수와 평균 사용량 간 상관관계는 거의 없음
- 공공 Wi-Fi 누적 설치 수와 인터넷 평균 사용 시간은 강한 양의 상관으로 관찰됨
- 인구 밀도와 설치 수 간에는 뚜렷한 양의 상관관계가 없으며, 일부 지역은 설치 수는 많으나 사용량이 낮음
